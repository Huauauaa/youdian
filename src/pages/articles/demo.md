# 有点：一篇 Demo 文章

欢迎来到 Youdian 的第一篇示例文章。这里用一段轻量的内容展示
Astro Markdown 页面如何作为文章发布，并作为后续内容样式和信息结构
的起点。

## 为什么需要这篇 demo

一个最小可用的文章 demo 可以帮助我们快速验证几件事：

- 页面是否能通过 GitHub Pages 正确访问
- 中文标题、段落和列表的排版是否自然
- 首页是否能清楚地引导读者进入文章

## 可以继续扩展什么

后续可以在这个基础上加入文章列表、发布日期、标签、作者信息以及
统一的内容布局。等内容变多之后，也可以把文章抽象到内容集合中，
让首页自动读取并展示最新文章。

现在，这篇 demo 已经可以作为一个简单的内容页面被访问了。
